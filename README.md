# AI SEO Tools Hub

智能SEO工具集 - 基于AI的现代SEO优化平台

## 可用域名

- **推荐**: `ais-seotools.com` ✅ 可用
- 备选: `aitools-seo.com` ✅ 可用

## 服务器配置

- Dokploy: http://localhost:3000
- Uptime Kuma: http://localhost:3001

## 技术栈

- Node.js 20+
- Docker & Docker Compose
- Dokploy (部署管理)
- Uptime Kuma (监控)

## 项目结构

```
.
├── apps/
│   ├── web/           # Next.js 前端
│   └── api/           # API服务
├── packages/
│   └── shared/        # 共享代码
├── docker-compose.yml
└── README.md
```

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/ddy4633/aiseo-tools-hub.git
cd aiseo-tools-hub

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```
